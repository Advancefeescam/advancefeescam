## Background

`miro` development stalled for a while on broken Xiaomi source releases — incomplete blobs, missing hardware baseline data, trees that didn't build clean out of the box. Reconstructing a working hardware baseline from scratch, bypassing the broken upstream pieces entirely, is what unblocked it. Both ROMs above are built on that fix.
